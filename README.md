## 📌 Overview

This is my personal portfolio website — a single-page experience built to present who I am, what I can do, and the projects I've built. Designed with clean HTML, CSS, and JavaScript, the site is fully responsive and hosted on GitHub Pages.

---

## 🗂️ Sections

| Section | Description |
|---------|-------------|
| 🏠 **Home** | A welcoming hero section with a brief introduction |
| 👤 **About** | A short bio about me and my background |
| 🛠️ **Skills** | Technologies and tools I work with |
| 💼 **Experience** | Work history, internships, and roles I've held |
| 🚀 **Projects** | A showcase of my personal and academic projects |
| 📬 **Contact** | Ways to get in touch with me |

## 🚀 Getting Started

### View Online

The easiest way — just open the live site:
👉 **[jishnu314.github.io/in](https://jishnu314.github.io/in/)**

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/jishnu314/in.git

# 2. Navigate into the project folder
cd in

# 3. Open in your browser
open index.html
```

> No build tools, frameworks, or dependencies required. It's plain HTML, CSS, and JS — just open and go.
